# 1
Here is my control flow:

```mermaid
graph TD
    A(开始) --> B[设置目标风速 (Setpoint)];
    B --> C[风速计测量当前的“实际风速”];
    C --> D{实际风速 == 目标风速?};
    
    D -- 是 --> C;
    
    D -- 否 --> E{实际风速 < 目标风速?};
    
    E -- 是 --> F[控制器增加PWM信号];
    E -- 否 --> G[控制器降低PWM信号];
    
    F --> H[风扇提高转速];
    G --> I[风扇降低转速];
    
    H --> J(芯片周围风速改变);
    I --> J;
    
    J --> C;
